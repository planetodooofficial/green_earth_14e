Import Product Image

This module allows you to import images for a product/product variant from a CSV file using URL and file path.

INSTRUCTION TO USE THIS MODULE

  1. Go to Sales menu
  2. Go to Products Menu -> Products
  3. Click on Action Menu -> Import Products

  4. Wizard will get open then click on Upload File
  5. Click on Import Products

    6. Please import excel file with .csv extension


  6. Import Multiple Image Through CSV File

    7. Note:-Product should already available in system with internal reference

    Note:-You can import only those product's images which internal reference already available in the system
